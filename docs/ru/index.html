Lineage 2 JavaScript клиент

Идея состоит в том, чтобы иметь клиентскую библиотеку Lineage 2, которая позволяет другим проектам создавать клиентские функции L2, ботов, игровых помощников или тесты

TypeScript

Использование новейших языковых функций обеспечивает структуру для создания сложных приложений, но достаточно гибко, чтобы не мешать.

Прогрессивный

Использует преимущества новейших функций JavaScript, привнося шаблоны проектирования и зрелые решения в Node.js мир.

Разносторонний

Адаптируемая экосистема, которая является полноценной основой для всех видов серверных приложений.
